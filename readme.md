# Lesson 12. Create Landing Page via SCSS and best practices

### Installation
Download [node.js](https://nodejs.org/en)

### Launch

Copy 2 files
* gulpfile.js
* package.json

Also need to add **sass** folder and **index.html** file in root directory

Install packages
```
npm i
```
Launch compiler and server
```
npm run serve
```
